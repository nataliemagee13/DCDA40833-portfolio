Hometown Map

Lab 06: Interactive Map of Meaningful Locations in Fort Worth

Overview

This project creates an interactive map of meaningful locations in Fort Worth using Python, Folium, and Mapbox. The map displays 10 important places that are part of my daily life, from where I study to where I work and spend time with friends.

Interactive Map

Click on any marker to see the location name. Zoom in and out to explore different parts of Fort Worth.

Reflection

For the original base map design I wanted to start out with something simple and clean, so I used colors that you would typically see on a map like green and blue. I also looked at references of soft colors and tried to limit the clutter on visuals. I was inspired by maps that used subtle tones and where the points of interest would be the main focus. I also tried to adjust the visibility for major roads so they would be clearer when zoomed in. I think one of the bigger challenges in this lab was designing a map that would be able to zoom in and out without being too difficult to read. To help combat this, I prioritized major roads and geographic figures when zoomed out so users could still understand the overall layout of the city. The technical part of this lab was difficult for me because it involved using Python to generate an interactive map using Folium, this was hard because I had to configure the environment in VS code which required different libraries like pandas and folium. I also struggled with correctly integrating my Mapbox style into the code. I used AI such as CoPilot to assist in creating my python code and page structure. Overall,l this project really helped me to understand how design decisions and technology like Mapbox and Python work together when creating interactive maps.
