Hometown Map

Lab 06: Interactive Map of Meaningful Locations in Fort Worth

Overview / Lab Requirements

For this lab, I designed an interactive hometown map in Fort Worth using Python, Folium, and Mapbox. My goal was to create a map that looked simple and clean while still being easy to read at different zoom levels.

  • Map a set of personally meaningful hometown locations.
  • Use Python and Folium to generate an interactive web map.
  • Apply map styling choices that support readability and visual clarity.
  • Reflect on design decisions and technical setup challenges.

Visual / Embedded Content

Click any marker to see the location name, and zoom in or out to explore different parts of Fort Worth.

What I Learned

I learned that making a good interactive map is both a design and technical process. The visual side mattered more than I expected, especially with color choices, clutter, and what stays visible when users zoom in and out. I also learned that setup in VS Code, Python libraries, and Mapbox styling all have to work together for the map to function correctly.

What This Demonstrates About My Abilities

This lab demonstrates that I can make thoughtful design decisions while building a technical project. It shows that I can troubleshoot Python workflow issues, configure required tools, and use AI support to help build and organize code when working with unfamiliar mapping technology.

Reflection

Design Decisions

For the original base map design, I wanted to start with something simple and clean, so I used colors you would typically see on a map, like green and blue. I also looked at references with soft colors and tried to limit visual clutter. I was inspired by maps with subtle tones where points of interest are the main focus. I also adjusted the visibility for major roads so they would be clearer when zoomed in.

Technical Challenges

One of the bigger challenges in this lab was designing a map that could zoom in and out without becoming difficult to read. To help with that, I prioritized major roads and geographic features when zoomed out so users could still understand the overall city layout. The technical side was also difficult because I had to generate the map in Python using Folium, configure my environment in VS Code, and work with libraries like pandas and folium. I also struggled with correctly integrating my Mapbox style into the code.

Overall

I used AI tools such as Copilot to assist with parts of my Python code and page structure. Overall, this project helped me understand how design decisions and technology like Mapbox and Python work together when creating interactive maps.
